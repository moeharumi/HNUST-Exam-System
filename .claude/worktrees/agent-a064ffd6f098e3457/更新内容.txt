HNUST仿真平台 更新内容 (v1.0.4 → v1.0.5)
===========================================

【重大重构】
- 整个项目从单文件架构 (exam_system.py, 2669行) 重构为标准 MVC 分层包结构 (hnust_exam/)
  - models/     — 数据模型层 (Exam, Question, Result)
  - services/   — 业务服务层 (评分引擎、配置管理、备份管理、Python环境检测、更新检查)
  - views/      — 视图层 (欢迎页、选卷页、答题页、结果页、导航栏、各类对话框)
  - utils/      — 工具层 (常量、主题管理、辅助函数、UI帮助)
  - viewmodels/ — 视图模型层 (预留 MVVM 扩展)
- 入口文件从 exam_system.py 改为 main.py

【UI 框架升级】
- GUI 框架从 tkinter 迁移至 PySide6 (Qt for Python)
- 全新的 QSS 全局样式表，统一控件外观（按钮、输入框、滚动条、进度条等）
- 支持高 DPI 屏幕（Per-Monitor DPI Awareness）

【新增功能】

1. 深色模式
   - 新增完整的深色主题配色方案（30+ 颜色变量），可在设置中切换

2. 设置面板
   - 新增设置对话框，包含 5 个设置卡片：
     · 个人信息（姓名、学号）
     · 即时反馈模式（答题后立即显示对错）
     · 深色模式开关
     · 字体大小调节（80%-150% 滑块，实时预览）
     · 考试标记管理（查看标记图例、清除所有标记）

3. 即时反馈模式
   - 新增设置项：答题后立即显示正确/错误状态
   - 单选/判断题选中后即时变色（绿色=正确，红色=错误）

4. 考试进度追踪
   - 自动记录每套试卷的答题进度到 progress.json
   - 选卷页面显示完成状态：✓ 85%（已完成）、○（进行中）
   - 鼠标悬停显示：最后完成日期、最高分、最近开始日期
   - 答题页面提交后自动保存进度和最佳成绩

5. 题目分析功能
   - 答题页新增"分析"按钮，自动复制当前题目到剪贴板并打开豆包(doubao.com)辅助分析

6. 试卷选择页增强
   - 新增卡片式弹性弹出动效（AnimatedCardDelegate）
   - 自定义 ToggleListWidget 支持惯性滚动和指数缓动
   - 设置按钮直达设置面板

7. Python 环境检测对话框
   - 新增独立对话框：当存在程序题但未找到 Python 时弹出
   - 提供"自动搜索"、"手动选择"、"下载 Python"、"跳过"四个选项

8. 答题结果页增强
   - 大号成绩卡片，颜色分级（绿色≥90%/蓝色≥60%/红色<60%）
   - 自定义进度条显示得分比例
   - 详情表格显示每题的用户答案和正确答案
   - "查看"按钮打开单题回顾对话框（ReviewDialog）

9. 提交确认页增强
   - 统计卡片（总数/已答/未答/标记数）
   - 黄色警告框提示未答的标记题
   - "复制未答题号到剪贴板"按钮
   - "检查标记题目"按钮快速跳转

10. 更新提示对话框
    - 显示版本变更信息、发布日期
    - 滚动显示更新日志
    - "跳过此版本"、"稍后"、"立即更新"三个选项

11. 自定义控件
    - ToggleSwitch：带动画的开关切换控件（200ms 滑动动画）
    - ArrowIndicator：带旋转动画的折叠箭头
    - SmoothScrollArea：三次方缓动的平滑滚动区域

【动画与交互改进】
- 导航面板：折叠/展开动画、交错渐入效果、动画代数防抖
- 选卷列表：60fps 弹性弹出动画、动态行高扩展
- 欢迎页：10 秒倒计时进入按钮
- 全局鼠标滚轮惯性滚动 + 边缘弹性回弹（修复了旧版重复实现的问题，统一为可复用组件）

【评分引擎改进】
- 新增 difflib.SequenceMatcher 模糊匹配（相似度≥0.9 视为正确），用于程序题评分
- 新增引号归一化处理（单引号统一转双引号后再比较）
- 填空题评分拆分逻辑统一支持中英文逗号和分号
- 评分逻辑从主程序中抽离为独立 grader 服务模块

【架构改进】
- 新增 ConfigManager：线程安全的 JSON 配置管理（~/.hnust_exam/），支持配置文件、进度文件、跳过版本文件
- 新增 BackupManager：独立的程序文件备份/恢复/清理服务
- 新增 UpdateChecker：基于 Qt Signal 的异步 GitHub 更新检查，线程安全回调到主线程
- Python 环境检测抽离为独立服务模块
- 全局异常捕获：未处理异常自动写入 ~/Documents/HNUST_Exam_Logs/ 崩溃日志
- 所有硬编码常量（考试时间、题型顺序、版本号等）集中到 constants.py

【测试】
- 新增 pytest 测试框架
- ConfigManager 测试：5 个用例（默认值、存取、未知键保留、进度空/存取）
- Grader 测试：17 个用例
  · 填空题评分（8个）：精确匹配、大小写、分号/逗号分隔、数量不匹配、空白处理
  · 程序题评分（5个）：精确匹配、空白归一化、引号归一化、多行、模糊匹配
  · 完整评分流程（4个）：单选全对、判断题中文映射、填空多空、未作答

【项目工程化】
- 新增 requirements.txt 明确依赖：PySide6>=6.6.0, pandas>=2.0.0, openpyxl>=3.1.0, requests>=2.31.0, Pillow>=10.0.0, pytest>=8.0.0
- 新增 项目说明.txt（723行），包含完整的项目结构说明、各文件修改指南、常见修改教程、运行/测试/打包指南
- PyInstaller 打包配置更新：入口改为 main.py，添加 PySide6 收集钩子和隐藏导入，指定临时目录

【版本】v1.0.4 → v1.0.5
