## 📌 软件介绍

本软件是模仿学校机房万维考试系统开发的**免费练习工具**。
- 专为 HNUST 同学设计，完美还原考试界面和操作流程，让你提前熟悉考试环境。
- 内置多套模拟题库，支持自动判分，是你备考的得力助手。

> ⚠️ **重要提示**：
> 如若想使用**程序设计、程序改错、程序填空**三种功能，你的电脑需预先配置好 Python 运行环境。

## 💻 功能特点

- ✅ **题型全面**：支持单选、判断、填空、程序填空、程序改错、程序设计等多种题型
- ✅ **自动判分**：即时显示答题结果、正确答案及详细解析
- ✅ **一键编辑**：一键打开程序文件，自动调用 IDLE 进行代码编辑
- ✅ **智能备份**：程序文件自动备份，支持一键重置，不怕改错
- ✅ **题目导航**：侧边栏快速跳转到未答或标记的题目
- ✅ **考试计时**：全真模拟考试倒计时，时间到自动交卷
- ✅ **自动更新**：软件内置更新检测，第一时间获取最新版本

## 🐍 Python 环境配置指南（仅编程题需要）

如果你只做选择题和判断题，可以跳过此节。如需使用编程题功能，请按以下步骤配置：

1.  **检查是否已安装**：
    打开 CMD（命令提示符），输入 `python --version` 并回车。
    - 如果显示 `Python 3.x.x`，说明已安装。
    - 如果提示“不是内部或外部命令”，请继续往下看。

2.  **下载与安装**：
    - 访问 [Python 官网](https://www.python.org/downloads/) 下载最新版 Windows 安装包。
    - **关键步骤**：安装时务必勾选 **"Add Python to PATH"**！
    - 点击 "Install Now" 完成安装。

3.  **配置 IDLE（可选）**：
    软件默认调用 IDLE 打开代码文件。只要安装了 Python，IDLE 通常会自动关联。
