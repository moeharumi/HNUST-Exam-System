## 🐍 编程环境配置指南（仅编程题需要）

如果你只做选择题和判断题，可以跳过此节。如需使用编程题功能，请根据语言按以下步骤配置：

### Python 环境

1.  **检查是否已安装**：
    打开 CMD（命令提示符），输入 `python --version` 并回车。
    - 如果显示 `Python 3.x.x`，说明已安装。
    - 如果提示”不是内部或外部命令”，请继续往下看。

2.  **下载与安装**：
    - 访问 [Python 官网](https://www.python.org/downloads/) 下载最新版 Windows 安装包。
    - **关键步骤**：安装时务必勾选 **”Add Python to PATH”**！
    - 点击 “Install Now” 完成安装。

3.  **配置 IDLE（可选）**：
    软件默认调用 IDLE 打开代码文件。只要安装了 Python，IDLE 通常会自动关联。

### C 语言环境（MinGW-w64）

1.  **下载 MinGW-w64**：
    - 访问 [MinGW-w64 官网](https://www.mingw-w64.org/) 或 [WinLibs](https://winlibs.com/) 下载。
    - 推荐下载 **UCRT Runtime** 版本。

2.  **安装与配置**：
    - 解压到目录（如 `C:\mingw64`）。
    - 将 `C:\mingw64\bin` 添加到系统环境变量 **PATH** 中。

3.  **验证安装**：
    打开 CMD，输入 `gcc --version` 并回车，显示版本信息即为成功。
